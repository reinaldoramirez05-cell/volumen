# Volumen | First Bilingual Rock Magazine in the US

Volumen is a bilingual rock magazine based in Austin, Texas. This digital experience explores live music photography, artist interviews, and the pulse of rock en español and indie music.

![Volumen Showcase](https://volumen.media/photos/wide-stage-view.jpg)

## 🎸 Project Overview

This repository contains the interactive web experience for Volumen Magazine. It features a heavy emphasis on visual storytelling and interactive 3D elements to immerse users in the rock scene.

### Key Features
- **Bilingual Interface**: Fully accessible in both English and Spanish.
- **Interactive 3D Landing**: A Spline-powered 3D environment for a premium first impression.
- **Responsive Media**: Curated photography galleries and editorial content optimized for all devices.
- **SEO Optimized**: Complete metadata, semantic HTML, and automated sitemap generation.

## 🛠 Tech Stack
- **Structure**: Semantic HTML5.
- **Styling**: Vanilla CSS3 with custom properties and modern layouts.
- **Logic**: Vanilla JavaScript (ES6+).
- **Interactive**: [Spline Runtime SDK](https://spline.design/) for 3D interactions.
- **Hosting**: Deployed on [Netlify](https://www.netlify.com/).

## 🚀 Deployment & Clean Routing

The project is configured for deployment on Netlify using the included `netlify.toml`.

### Clean URLs
Navigating to the following paths will automatically render the correct pages:
- `/home` -> Main magazine feed
- `/editorial` -> Editorial content
- `/interviews` -> Artist interviews
- `/events` -> Live coverage
- `/contact` -> Collaboration pitches

### Automated Performance Tracking
Lighthouse scores for Performance, Accessibility, Best Practices, and SEO are automatically tracked on each build via the Netlify Lighthouse plugin.

## 📈 Future Roadmap (Optimization)
While currently a pure HTML/CSS/JS project, future migrations could include:
- **Framework Migration**: Moving to **Astro** or **11ty** for better asset optimization (image compression, JS bundling).
- **Dynamic Content**: Integrating a headless CMS for easier editorial updates.
